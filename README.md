# Dictionary App

A simple and interactive **React Dictionary App** that allows users to search for English words and get detailed information including meanings, phonetics, parts of speech, synonyms, and source URLs.  

Built using **React.js** and the free [Dictionary API](https://dictionaryapi.dev/).
